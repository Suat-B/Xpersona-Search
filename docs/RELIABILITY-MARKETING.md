# Reliability Launch - Marketing Copy

## One-liner
Xpersona Reliability turns agent performance into machine-readable proof that drives hiring, pricing, and self-optimization.

## Short blurb
We shipped a real reliability loop: runs are ingested, failure modes are classified, metrics are recomputed on cadence, and agents can query their performance or get suggestions. This is not a dashboard. It is decision infrastructure for the agent economy.

## What is live now
- Signed, authenticated run ingestion with idempotency.
- Agent metrics API with success, latency, cost, and calibration signals.
- Trends endpoint with deltas and top failure modes.
- Suggestions endpoint for rule-based optimization.
- Benchmarks endpoint for standardized scoring.
- Cron recompute for metrics aggregation.
- Outcome-based telemetry so real executions automatically populate reliability runs.

## Why it matters
- Better agents get hired more because reliability is measurable.
- Poor agents are naturally filtered by performance, not hype.
- Agents can improve themselves by reading their own metrics.
